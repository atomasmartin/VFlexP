The grid structure is composed of three tables: one for the buses and one for the lines. The buses table contains the following columns:
  • bus: bus number
  • P_kW_load: active power consumed at that bus in kW
  • Q_kVAr_load: reactive power consumed at that bus in kVAr
  • C_pu_load: capacitive at that bus in pu
  • P_kW_gen: active power generated at that bus in kW (read by PF if bus type is not 3, slack)
  • Q_kVAr_gen: reactive power generated at that bus in kVAr (read by PF if bus type is 1, PQ fixed)
  • DGs: type of generator (GFr, GFl, SG, SC, IG, DFIG)
  • v_mod: voltage magnitude of that bus in pu (read by PF if bus type is 2 or 3)
  • v_angle: voltage angle in rad (completed after PF)
  • Sb_kVA: base power of the system in kVA (only first column)
  • PQ_gen: type of generator (1 is fixed P and Q, 2 is fixed P and v, 3 is fixed v and slack P and Q)
  • Complexity_bus: complexity of the bus (static omega, static, dynamic, only R)
  • Complexity_load: complexity of the load (static omega, static, dynamic, PQ, dynamic PQ)
  • Complexity_bus_LF: complexity of the bus in load flow (static omega, static, dynamic, only R)
  • Complexity_load_LF: complexity of the load in load flow (static omega, static, dynamic, PQ, dynamic PQ)
