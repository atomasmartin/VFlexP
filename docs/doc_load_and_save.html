The tool allows you to load and save bus and line data from an Excel file.

Loading Grid Data from an Excel File

The data format for loading is based on MATPOWER-FUBM. These tables can include variables that must be defined in MATLAB's base workspace.

Saving Grid Data to an Excel File

The tool allows you to save bus and line data to an Excel file. The file will contain two sheets: "Buses" and "Lines". Any existing data in the cells that are being written to will be overwritten.
