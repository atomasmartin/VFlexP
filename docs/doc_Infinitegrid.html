Infinite Grid

The infinite grid is set as an ideal voltage source that imposes a fixed voltage and frequency at its point of connection. However, it can be easily modified to include a droop effect that reduces its strength, by: $$ \omega = \omega_{ref} - m_P (P-P_{ref}) $$ $$ v = v_{ref} - n_Q (Q-Q_{ref}) $$

Droop effect

MATLAB Code 
        
          % Simulink model of an infinite grid
          droop_coefficient = 0.1; % Droop coefficient
          set_param(block_path, 'droop_coefficient', droop_coefficient);
